## Debian Package File Conventions
`PACKAGE-NAME-VERSION-RELEASE_ARCHITECTURE.deb`
- `ARCHITECTURE`
    - `amd64`
    - `all`

`apt-get download vim`, Downloads debian package file

## The dpkg Command Set
Installing, updating, & removing `.deb` package files. 

||||
|---|---|---|
|`-c`|`--contents`|To view |
|`-C`|`--audit`||
|N/A|`--configure`||
|N/A|`--get-selections`||
|`-i`|`--install`|Installs the package but if installed it upgrades the package instead.|
|`-I`|`--info`||
|`-l`|`--list`|**Status codes**<ul><li>`i`, Installed</li><li>`u` Unpacked, not configured, which is a problem.</li></ul>|
|`-L`|`--listfiles1`||
|`-p`|`--print-avail`||
|`-P`|`--purge`|Removes the package completely|
|`-r`|`--remove`|Removes the package but keeps any configuration and data files.|
|`-s`|`--status`||
|`-S`|`--search`||

## Looking at the APT suite
**Advanced Package Tool (APT)**
- `apt-cache`, Information about the package database

**Automatic security upgrades**
- Edit, `/etc/apt/apt.conf.d/10periodic` file
    - `APT::Periodic::Update-Package-Lists`
        - `0`, To turn it off
        - `1`, To turn it on
- `man unattended-upgrade`, To learn more about **Automatic security upgrades**.
